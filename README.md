# 📈 山东省城乡居民养老保险精细测算器

> 一个基于 HTML5 + Vanilla JS 的单文件养老金测算工具，专为山东省城乡居民养老保险政策定制。

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-Web-green.svg)

## 📖 项目简介

这是一个轻量级、响应式的网页计算器，旨在帮助用户通过精细化的参数配置，估算未来退休后的养老金领取金额。

不同于通用的粗略计算器，本项目针对**山东省**的具体政策（如长缴激励机制、高龄基础养老金倾斜、政府缴费补贴分档等）进行了适配，支持“固定金额”和“逐年设定”两种规划模式，并提供可视化的图表分析。

## ✨ 核心功能

* **📱 移动端优先设计**：精美的 UI 界面，完美适配手机、平板和桌面端。
* **🧮 双模式测算**：
    * **简易模式**：假设未来每年按固定档位缴费。
    * **精细模式**：可针对未来每一年的经济状况，单独设定个人缴费和子女资助金额。
* **📊 可视化图表**：
    * **饼图**：清晰展示基础养老金与个人账户养老金的构成比例。
    * **折线图**：展示退休后累计领取金额走势，并自动计算**回本周期**。
* **⚙️ 深度参数配置**：
    * 支持自定义基础养老金标准、长缴激励费率、账户年化利率。
    * 内置山东省政府缴费补贴规则（350档补30元/500档补50元/800及以上补80元）。
    * 自动计算高龄倾斜（65-74岁加发5元，75岁以上加发10元）。
* **🔒 数据安全**：纯前端实现，所有计算均在本地浏览器完成，不上传任何用户数据。

## 🧠 测算逻辑说明

本计算器严格遵循以下公式进行估算：

1.  **个人账户总额** = ∑ (历年个人缴费 + 历年政府补贴 + 历年集体补助) × (1 + 年利率)^年限
2.  **月个人账户养老金** = 个人账户总额 ÷ 139 (60岁退休计发月数)
3.  **月基础养老金** = 当前基础标准 + 长缴激励 + 高龄倾斜
    * *长缴激励*：缴费年限超过15年的，每多缴1年，基础养老金每月增加 `3元` (默认配置)。
    * *高龄倾斜*：65-74岁每月加发 `5元`，75岁及以上每月加发 `10元`。
